HTML5

A linguagem HTML (sigla de HyperText Mrkup Language - Linguagem de Marcação de Hipertexto) é responsável pela riação de documentos para a web, as chamadas páginas. Elas são divulgadas na WWW (World Wide Web) - Teia de Alcance Mundial.

Por meio dessa linguagem, você pode especificar atributos para um texto (como fonte, tamanho, cor, etc), e fazer os Hyperlinks que permitem navegar pela web. para isso, ela utiliza marcações, chamadas tags, no ponto da página em que é feito uma liação com outra.

Portanto, o HTML é um conjunto de tags utilizadas para edição de uma página de internet. Tudo isso, feito através de uma linguagem fácil de se trabalhar
